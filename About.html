Sobre mim

Olá. Me chamo Thiago dos Santos Rodrigues e estou fazendo a formação da Alura em Front-End. Tem sido muito desafiador e percebo que tenho muito o que melhorar nesse processo. Até o presente momento eu estou aprendendo as tecnologias HTML (linguagem de marcação) e CSS (folha de estilo), não se tratam de linguagens de programação ainda, nesse quesito pretendo aprender JavaScript e suas tecnologias (React, node JS) e lógica de programação aplicada ao JS.

Falando das tecnologias estudadas até o momento, HTML tem sido bem tranquila, visto que é apenas uma linguagem de marcação, não tenho grandes dificuldades em marcar uma página de forma funcional e semântica com base no meu nível de iniciante. Já em se tratando do CSS eu ainda tenho bastante a evoluir, FlexBox por exemplo é uma ferramenta onde eu vou dar uma atenção especial nos próximos meses para garantir o meu domínio sobre ela. Também tenho que aprender sobre Grid layout que é uma forma de posicionamento complementar.

Outras ferramentas que ou aprendi foram, variáveis css que é uma forma prática de deixar o css mais otimizado no quesito mudanças futuras e o Media query que permite uma condição específica para a adaptação de conteúdo (importantíssima para um site responsivo). No mais meu caminha passará por me aperfeiçoar nessas tecnologias para dar início ao aprendizado de JavaScript tendo assim uma maior condição de oferecer meus serviços e resolver problemas de determinadas empresas, e ou pessoas.

Importante: Essa página foi desenvolvida no dia 23/02/2023, ficando claro que minhas habilidades atuais valem apenas para a presente data visando que estarei em constante aprendizado nos dias, meses e anos que virão.
Grato!
